# AMU Connect - Demo Credentials

## Student Login Flow

All demo credentials use the **Admission Verification Flow**:
1. Go to **Create Account** page
2. Select **Student** role
3. Enter the Registration Number below
4. Click **Verify Admission**
5. The system will auto-fill all details
6. Complete the registration
